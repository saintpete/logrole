Participants

Despite offering this functionality in the Dashboard, Twilio does not offer an API to view conference participants, or a way to figure out which call sids are involved in a call. Please contact Customer Support and ask for the ability to retrieve call SIDs for a Conference via the API.

Auto-scraping this data from the Twilio Dashboard would be a violation of the Acceptable Use Policy. We cannot embed it via an <iframe>, because Twilio sets a X-Frame-Options: SAMEORIGIN framebusting header. Please contact Support to ask for the ability to view conference Participants after a conference has finished.

If you have access to the Twilio Dashboard, you can click here to view the Participants for this conference.
